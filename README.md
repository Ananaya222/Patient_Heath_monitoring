#  Patient Health Monitoring System

##  Overview

This project is an IoT-based health monitoring system that continuously monitors a patient's vital signs in real time. It collects health data from sensors and displays the readings to help monitor a patient's condition.

---

##  Features

- Monitors heart rate, body temperature, and oxygen (SpO₂) levels.
- Displays real-time health data.
- Uses IoT sensors for continuous patient monitoring.
- Helps in early detection of abnormal health conditions.

---

##  Technologies Used

- ESP32
- MAX30100 Pulse Oximeter Sensor
- DS18B20 Temperature Sensor
- DHT11 Sensor
- Arduino IDE
- Embedded C

---

##  Working

- The sensors collect the patient's health data.
- ESP32 reads the sensor values.
- The collected data is processed and displayed in real time.
- The system continuously monitors the patient's health parameters.

##  How to Run

1. Connect the sensors to the ESP32.
2. Open the project in Arduino IDE.
3. Install the required libraries.
4. Upload the code to the ESP32.
5. Monitor the sensor readings using the Serial Monitor.
